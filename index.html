Que.1: "Problem:" 
            You are given a string (STR) of length N, consisting of only the lower case English alphabet.
            Your task is to remove all the duplicate occurrences of characters in the string.

            Input:
            abcadeecfb

            Output:
            abcdef

Ans: 
function removeduplicate(inputstr)
            {
                var chars = new Set(); 
                var result = [];   //array
            
               for( const char of inputstr) 
                if(!chars.has( char))
                {
                    chars.add(char);
                    result.push(char);
                }
                return result.join('');
            }
            var STR = "abcadeecfb";
            console.log(removeduplicate(STR))

        Reason: Here, we are creating a function and passing string as agrs in function using var STR
        and args take as parameter inside function using Set() we can find unique character and store in result i.e. array.
        Using for of() check for each characters of string if condition check each character of inputstr is not available in
        chars then add that character in set i.e. chars and push into array.and after completing for all input character convert
        array into string using join() method and return the output i.e. abcdef
